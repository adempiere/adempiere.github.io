# Cash Journal


Cash transactions

Help
## Help

The Cash Journal Window is used to record disbursements from and receipts to Petty Cash

Window Type
### Window Type

**Transaction**

:::tip
The Sales Transaction checkbox indicates if this item is a Sales Transaction.
